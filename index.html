Projects

1.AI Model for Posture Correction

Slouchometer is an AI-powered computer vision system designed to monitor and correct poor posture in real time using a standard webcam. The system leverages pose estimation techniques to analyze upper-body landmarks and classify user posture as either correct or slouched. Upon detecting sustained slouching, the model provides timely alerts to encourage posture correction, promoting long-term ergonomic health.

The application continuously logs posture data and stores it in structured formats for analysis. This data is further utilized to build visualization dashboards that help users track posture trends, identify problem patterns, and measure improvement over time. Slouchometer is a non-intrusive, cost-effective solution suitable for students, professionals, and remote workers, achieving high accuracy while running on commonly available devices.

2. AI Based Learning Engagement Analyzer

This project focuses on enhancing the effectiveness of online education by analyzing student engagement using AI models. The system captures live video feeds during online classes and applies computer vision and machine learning models to evaluate facial cues such as attention, distraction, and emotional states like confusion or interest. The engagement metrics are processed in real time and integrated with a custom Learning Management System (LMS). Instructors gain access to analytical dashboards that provide both individual and class-level engagement insights, enabling data-driven instructional decisions. The solution addresses a critical gap in online learning by moving beyond attendance tracking to actual attentiveness and behavioral analysis.

3. Pothole Detection Tracking System

This project implements a real-time pothole detection system using the YOLOv8 object detection framework to support smart infrastructure monitoring and safer road conditions. The model is custom-trained on a labeled dataset containing both images and videos, enabling accurate and reliable pothole identification in real-world scenarios. The system delivers high-performance detection with a precision of 98.4% and a mAP@0.5 of 94%, ensuring minimal false positives while maintaining strong detection coverage. It supports both image-based and live video inference, making it suitable for deployment in vehicle-mounted cameras, surveillance systems, or city-wide road monitoring solutions. The implementation follows a modular and scalable design, with dedicated components for training, validation, and inference. Real-time bounding boxes with confidence scores are overlaid on detected potholes, allowing clear visual interpretation. The solution is designed for easy deployment and rapid testing, enabling efficient integration into automated road inspection and maintenance workflows.
